****cause: 
psycopg2 est le driver PostgreSQL classique (bloquant).

Pour l’async avec SQLAlchemy, il faut utiliser un driver compatible, comme asyncpg.

Ton DATABASE_URL doit être adapté pour indiquer à SQLAlchemy d’utiliser asyncpg.

***********solution:
pip install asyncpg

Dans ton .env, il doit être au format :
DATABASE_URL=postgresql+asyncpg://user:password@host:port/dbname
